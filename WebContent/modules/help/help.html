Hilfeseite

Unidaplan ist eine Webbasierte Software um wissenschaftliche Proben zu verwalten. Es können Probentypen (z.B. Pulverproben, Solarzellen, Halbleiterlaser, usw.) definiert werden. Die Proben können bestimmte Parameter und Eigenschaften aufweisen. Die Auswahl und Definition der Parameter erfolgt über den Adminstrator.
